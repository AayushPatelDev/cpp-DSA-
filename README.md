# 🌳 Trees & Graphs in C++ – Interview Prep Repository

This repository contains my implementations of the **most common Trees and Graphs problems**, along with the **core concepts and fundamentals required to understand them** using **C++**.

## 📌 Purpose

The main goal of this repo is to:

* Practice and strengthen problem-solving skills
* Build a solid understanding of Trees and Graphs
* Prepare for technical interviews
* Maintain a structured collection of commonly asked questions

## 🧠 What’s Included

* Basic concepts and foundational implementations
* Common Tree problems:

  * Traversals (Inorder, Preorder, Postorder, Level Order)
  * Binary Search Trees (BST)
  * Depth/Height, Diameter, etc.
* Common Graph problems:

  * BFS (Breadth-First Search)
  * DFS (Depth-First Search)
  * Shortest path algorithms
  * Connected components and more
* Clean and readable C++ code for each problem

## 🛠️ Language Used

* **C++**

## 🎯 Who This Is For

* Students preparing for coding interviews
* Beginners learning Trees and Graphs
* Anyone looking to practice DSA in C++

## 🚀 How to Use

* Browse through folders/topics
* Study the concepts and corresponding implementations
* Run and modify code for better understanding
* Practice regularly to build confidence

## ⚠️ Note

This repository is intended for **learning and practice purposes only**. Problems included here are based on commonly asked interview questions.
